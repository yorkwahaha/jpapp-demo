JPAPP 助詞全關卡規劃表 v1
設計原則

先定三條，之後你改 levels.v1.json / skills.v1.json / game.js 就比較不會亂掉。

1. 一關只教一個核心用法

不是「一個助詞 = 一關」，而是：

が 的自然現象是一關

が 的強調主詞是另一關

に 的時間是一關

に 的對象又是另一關

2. 外層格式統一，內層例外保留

每題最後都統一成：

chinese

segments

answers

grammarTip

skillId

choices

但像：

雨が降る → 下雨

問題が起きる → 問題發生

私も学生です → 我也是學生
這些例外語感照樣保留。

3. 每 5 關做魔王複習

第 5 / 10 / 15 / 20 / 25 / 30 / 35 關都做複習 Boss。

第 1 章：最基本句子骨架
1. は：主題提示

公式：

A は B です

例型：

私は学生です

これは本です

中文提示規則：

A 是 B

2. が：自然現象／自動發生

公式：

N が V

例型：

雨が降る

問題が起きる

ドアが開く

中文提示規則：

優先「名詞先 + 狀態變化」

特例：

雨が降る → 下雨

雪が降る → 下雪

3. を：受詞

公式：

N を V

例型：

本を読む

ご飯を食べる

中文提示規則：

動詞 + 受詞

例如：

吃飯

看書

4. に：時間點

公式：

T に V

例型：

七時に起きる

明日に行く 這種避免，時間搭配要精選

中文提示規則：

時間 + 動作

例如：

七點起床

今天休息

5. 魔王複習關 1

範圍：

は / が(自然發生) / を / に(時間)

第 2 章：移動與附加資訊
6. へ：移動方向

公式：

Place へ 行く / 帰る

例型：

学校へ行く

日本へ帰る

中文提示規則：

去 + 地點

回 + 地點

7. も：也

公式：

A も B

N も V

例型：

私も学生です

雨も降る

空も曇る

中文提示規則：

主詞/名詞 + 也 + 後段

例如：

我也是學生

雨也下

天空也轉陰

8. の：所屬／修飾

公式：

A の B

例型：

私の本

日本の料理

中文提示規則：

A 的 B

9. と：並列

公式：

A と B

例型：

本とノート

犬と猫

中文提示規則：

A 和 B

10. 魔王複習關 2

範圍：

へ / も / の / と
並混前面基礎助詞

第 3 章：場所、存在、對象
11. で：動作場所

公式：

Place で V

例型：

教室で勉強する

家で寝る

中文提示規則：

在 + 地點 + 做某事

12. に：存在地點

公式：

Place に N が ある / いる

例型：

部屋に猫がいる

机の上に本がある

中文提示規則：

在某處有某物 / 某人

13. が：存在主體

公式：

N が ある / いる

例型：

猫がいる

本がある

中文提示規則：

有貓

有書
這關可與上一關做出功能差異：

に 重地點

が 重存在主體

14. に：對象／接受者

公式：

人 に V

例型：

先生に聞く

友達にあげる

中文提示規則：

對老師問

給朋友

15. 魔王複習關 3

範圍：

で(場所) / に(存在地) / が(存在主體) / に(對象)

第 4 章：時間範圍與共同動作
16. から：起點／來源

公式：

A から B

Time から V

例型：

家から学校へ行く

九時から始まる

中文提示規則：

從…

自…開始

17. まで：終點／截止

公式：

A まで

Time まで V

例型：

駅まで歩く

三時まで勉強する

中文提示規則：

到…

到…為止

18. と：共同動作對象

公式：

人 と V

例型：

友達と遊ぶ

家族と食べる

中文提示規則：

和誰一起…

19. で：工具／手段

公式：

N で V

例型：

箸で食べる

日本語で話す

バスで行く

中文提示規則：

用…

搭…

20. 魔王複習關 4

範圍：

から / まで / と(共同) / で(工具)

第 5 章：方向、到達、目的
21. に：到達點

公式：

Place に 行く / 来る / 帰る

例型：

学校に行く

家に帰る

中文提示規則：

去／回到…

22. へ vs に：方向 vs 到達

公式：

日本へ行く

学校に行く

中文提示規則：

做對比型題目
這關適合做「同一動詞下選助詞」

23. に：目的

公式：

Vます-stem に 行く

例型：

勉強しに行く

買い物に行く

中文提示規則：

去做…

去買東西

24. や：不完整列舉

公式：

A や B

例型：

本やノート

りんごやバナナ

中文提示規則：

A、B 等等

25. 魔王複習關 5

範圍：

に(到達) / へvsに / に(目的) / や

第 6 章：對比、限定、原因
26. は：對比

公式：

A は… B は…

例型：

肉は食べるが、魚は食べない

今日は行く

中文提示規則：

A 的話…

至於 B…

27. で：原因／理由

公式：

N / 理由 で …

例型：

病気で休む

雨で中止

中文提示規則：

因為…

由於…

28. も：全面包含／連…也

公式：

N も…

一円もない

例型：

子どももできる

一円もない

中文提示規則：

連…也…

…也…

29. だけ：只有

公式：

N だけ

例型：

一人だけ来る

水だけ飲む

中文提示規則：

只有…

30. 魔王複習關 6

範圍：

は(對比) / で(原因) / も(強調) / だけ

第 7 章：進階基礎
31. しか：只…（後接否定）

公式：

N しか + 否定

例型：

水しか飲まない

一人しか来ない

中文提示規則：

只有…（但語感比 だけ 更限制）

32. を：經過場所

公式：

Place を V

例型：

公園を歩く

橋を渡る

中文提示規則：

穿過…

走過…

33. に：變化結果

公式：

N / 形容動詞語幹 + に なる

例型：

医者になる

きれいになる

中文提示規則：

變成…

34. が：強調主詞／疑問焦點

公式：

だれが…

何が…

例型：

だれが来ますか

何が好きですか

中文提示規則：

誰…

什麼…

這個之後你改 code 很重要，我先一起定下來。

基本：A 是 B

2. が

基本：動詞 + 受詞

經過場所：穿過 / 經過

4. に

時間：時間 + 動作

存在地：在某處有…

對象：對 / 給

到達：到…

目的：去做…

變化結果：變成…

5. へ

去 / 朝向…

6. で

場所：在某處做…

工具：用…

原因：因為…

7. も

…也…

強調：連…也…

8. の

…的…

9. と
